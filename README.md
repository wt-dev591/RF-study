# RF-study

Qucs-Sを使って、RF回路・伝送線路理論の基礎を確認するための勉強ログです。

## 目的

反射係数、S11、インピーダンス整合、λ/4線路などを、手計算とシミュレーションを対応させて理解することを目的とします。

## 使用ツール

- Qucs-S
- GitHub
- 必要に応じて Python

## 学習テーマ

- 反射係数 / Reflection Coefficient
- 特性インピーダンス / Characteristic Impedance
- S11
- インピーダンス整合 / Impedance Matching
- λ/4線路 / Quarter-wave Transformer
- LPF
- LNA
